npm-init指令是用于对包进行初始化操作。
其中npm-init 会以交互式操作来询问操作者的需求
npm init 命令的作用是将当前文件夹初始化为一个『包』， 交互式创建 package.json 文件
package.json是包的必备文件，每个包都需要有对应的package.json文件

内容示例：
{
	"name": "1-npm", 		#包的名字
	"version": "1.0.0", 	#包的版本
	"description": "", 		#包的描述
	"main": "index.js", 	#包的入口文件
	"scripts": { 		    #脚本配置
		"test": "echo \"Error: no test specified\" && exit 1"
	},
	"author": "", 			#作者
	"license": "ISC" 		#开源证书
}

注意事项：
包的名称(name属性)不能使用中文或者大写字符，否则会出现错误。
因此如果使用文件名作为包的默认名字，则文件名也不能包含中文或者大写字符。
包的版本号的格式为 x.x.x （其中 x 为数字）也必须严格参照规则，否则会出现错误
package.json文件可以手动创建或者修改

其中也可以使用npm init -y 或者 npm init --yes 快速创建文件，此时均使用默认参数
